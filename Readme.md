# Flask Htop App

A simple Flask application that displays:
- Your name
- System user
- Current IST time
- Output of the `top` command (system usage)

## 🔗 Live Preview
[Click here to open the app](https://animated-giggle-69v64p67xprv3r7px-5000.app.github.dev/htop)

## ⚙️ Requirements
- Python 3.x
- Flask
- pytz

## 🚀 How to Run Locally
```bash
pip install flask pytz
python app.py
